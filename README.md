# Oceanbed-backend2.0

![Version](https://img.shields.io/badge/version-2.0.0-blue.svg?cacheSeconds=2592000)

> Oceanbed 2.0 serverless Backend API using AWS Lambda and FaunaDB

## Install

```sh
npm install
```

## Usage

```sh
npm run setup-schema
```

## ENV Variables

`secrets.yml`

```yml
FAUNADB_ENDPOINT: ""
FAUNADB_SECRET: ""
FAUNDB_TEMP_USER: ""
```

## Deploy to AWS

```sh
serverless deploy
```
